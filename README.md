# Macroeconomic-Forecasting-and-Risk-Monitoring-Dashboard-Evidence-from-FRED-MD
This project develops a macroeconomic forecasting and risk monitoring dashboard using the FRED-MD dataset. It applies time-series econometric methods such as ARIMA, Holt-Winters exponential smoothing, Vector Autoregression, and Principal Component Analysis to forecast key macroeconomic indicators and construct a composite risk index.

The project is organized as a Git-ready Python codebase with scripts, notebooks, outputs, figures, tests, and a Streamlit dashboard for interactive visualization. It is designed for reproducible econometric analysis and can be extended for central banking, commercial banking, development finance, and macroeconomic risk assessment.
